# The Y2K Bug Archive

An educational history-summative website about the Year 2000 computer problem,
with a playable audio overview, flip-card facts, a script-review checklist, and
a collection of research resources.

## Run it

Open `http://localhost:8000` while the local server is running, or open
`index.html` directly in a browser.

## Add your podcast

Use the **Load recording** control in the player and select an MP3, WAV, M4A,
or other browser-supported audio file. The uploaded recording replaces the
included narrated demo for that browser session.

The included demo narration text and flashcard facts are in `script.js`.
